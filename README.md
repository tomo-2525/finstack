## プロジェクトについて

### FinStackとは？

**FinStack** は単なる家計簿や税金計算ツールではなく、税金・投資・資産運用を総合的に管理するためのファイナンスアプリケーションです。

### 開発目的

- フロントエンド、バックエンド、データベース、インフラ、設計、テスト、運用など、学んだ IT技術を継続的にアウトプットするためのポートフォリオです。
